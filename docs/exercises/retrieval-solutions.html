Exercises » Boolean Retrieval » Solutions

This page presents solutions for the Boolean retrieval exercise. The main class for building the inverted index is edu.umd.cloud9.example.ir.BooleanRetrieval and a sample command-line invocation is as follows:

hadoop jar cloud9.jar edu.umd.cloud9.example.ir.BooleanRetrieval index data/bible+shakes.nopunc
