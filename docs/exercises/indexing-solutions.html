Exercises » Inverted Indexing » Solutions

This page presents solutions for the inverted indexing exercise. The main class for building the inverted index is edu.umd.cloud9.example.ir.BuildInvertedIndex and a sample command-line invocation is as follows:

hadoop jar cloud9.jar edu.umd.cloud9.example.ir.BuildInvertedIndex \
  data/bible+shakes.nopunc index 10

Command-line invocation for a simple program that performs the analysis asked in the questions:

hadoop jar cloud9.jar edu.umd.cloud9.example.ir.LookupPostings index data/bible+shakes.nopunc
